My Portfolio

Stop Procrastinating

Stop Procrastinating is a web application designed to help users manage their tasks and overcome procrastination. The app utilizes Express, Node.js, MongoDB, and React to provide an efficient and user-friendly task management experience. With Stop Procrastinating, users can organize their tasks, set reminders, and stay motivated to complete their work on time.

UTJA Website

I developed a website for my club, UTJA, using a combination of HTML and CSS. To ensure a well-structured and responsive design, I implemented a 12-column grid system, which allowed me to arrange and align content effectively for the user interface. Additionally, I integrated JavaScript to enhance the user experience, making the website more interactive and engaging for visitors.

Youtube Clone

This YouTube clone project is a website that imitates the appearance of the YouTube homepage. It was built using HTML and CSS, and it uses grid and flexbox layouts to arrange the page's elements, such as video previews and text, in a way that closely resembles the real YouTube homepage. This project aims to replicate the familiar YouTube experience, offering users a visually similar and user-friendly interface for browsing and viewing content.
